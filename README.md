# Final_Project_United_EWR_Departure_Delays_2018_2023
Analyze United Airlines departure delays at Newark (EWR) by comparing on-time performance, average delays, delay causes, and time of day patterns between 2018 and 2023.

## Purpose
The purpose of this project is to examine how United Airlinesâ€™ departure performance at EWR has changed over time and to identify the main factors contributing to flight delays.
